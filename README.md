# Coupon World v4.0

Premium deal discovery homepage for coupon-world.in.

## Files

- `index.html` — homepage structure
- `style.css` — premium design
- `app.js` — dynamic deal loading from `coupons.json`
- `coupons.json` — keep your existing file in GitHub root

## How to install

Upload these files to the GitHub repository root:

```text
index.html
style.css
app.js
coupons.json
```

Then commit and wait 1–3 minutes for GitHub Pages.

Hard refresh:

```text
Ctrl + F5
```

## Important

This homepage reads:

```text
coupons.json
```

from the same folder as `index.html`.
